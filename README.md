# AI 기반 개인 맞춤형 학습 계획 생성 시스템

> 단국대학교 소프트웨어개론 팀 프로젝트 | 2026년 1학기

## 프로젝트 개요

대학생 사용자를 위해 학습 계획 수립과 과제·시험 준비의 비효율 문제를 해결하는 AI 기반 개인 맞춤형 학습 계획 생성 시스템.
시험 일정과 강의자료를 바탕으로 학습 계획을 자동 생성하고, 핵심 내용을 요약해 학습 관리를 돕는다.

## 팀 구성

| 이름  | 역할    |
| --- | ----- |
| 방주원 | PM    |
| 김연재 | 분석가   |
| 김주함 | 개발자   |
| 김채원 | QA/보안 |
| 김예성 | 설계자   |
## 주요 기능

- 이메일·비밀번호 기반 회원가입
- 로그인 사용자 대상 강의자료 업로드
- 과목별 시험 날짜 입력
- 시험 일정 및 강의자료 기반 학습 계획 자동 생성
- 강의자료 핵심 내용 요약 제공
